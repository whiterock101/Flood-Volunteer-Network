# Flood-Volunteer-Network
A real-time community-driven tool for reporting and tracking road flooding conditions. Built to help residents navigate safely by crowdsourcing GPS-tagged water levels and landmarks.

🌍 Purpose
This application was created to provide a decentralized, immediate way for neighbors to help each other during heavy rains. By submitting a report, a volunteer's location is instantly mapped, allowing others to see which roads are passable and which are submerged.🛠️ How it WorksHTML/JavaScript Front-end: A lightweight mobile-friendly interface for volunteers.Google Forms Integration: Acts as the secure "bridge" to collect data.Google Sheets Database: A live-updating master list that generates Google Maps links for every report.GitHub Pages: Provides the secure $https$ hosting required for mobile GPS functionality.

📍 Regional Focus
Initially launched to support residents in Sharjah, UAE, but designed with a flexible architecture that can be adapted for any municipality.
